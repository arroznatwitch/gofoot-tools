🔄 Actualizar base CSV com o envio de equipas

Carrega o players.csv mestre e um ou vários .json de equipas (mesmo sem id). Cada jogador já traz o seu team_id, por isso podes seleccionar os ficheiros de várias equipas de uma vez. Atribui ID a quem não tiver, preenche os atributos/OVER de quem aparecer, e deixa o resto do CSV como está.

🏆 Ligas & Países

Progresso dos elencos carregados. Um clube conta como equipa válida quando tem 20+ jogadores com todos os atributos + OVER. As médias (OVR/ATA/MEI/DEF) são feitas pelo melhor 11 (1 G · 2 L · 2 Z · 4 meio · 3 ataque), não pelo plantel todo. Clica num país para esconder/mostrar as ligas. Importa o players.csv na aba Jogadores para veres o progresso.

⚠️ Painel de desenvolvimento. Estas ferramentas alteram a base de dados em massa. Não uses sem saber exactamente o que estás a fazer — faz sempre uma cópia do teu players.csv antes.

🔑 Regenerar IDs (.csv)

Gera um ID único novo para todos os jogadores importados do players.csv (resolve IDs repetidos/antigos) e descarrega o .csv com todas as colunas preservadas. Importa primeiro a base na aba Jogadores.
